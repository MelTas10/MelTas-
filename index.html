Политика конфиденциальности

MelTas Bot — Антискам База

Последнее обновление: 8 ноября 2025

1. Общие положения

Настоящая Политика конфиденциальности (далее — «Политика») определяет порядок обработки и защиты персональных данных пользователей бота MelTas (далее — «Бот», «Сервис»), разработанного для защиты пользователей Telegram от мошенников и скамеров.

Используя Бот, вы соглашаетесь с условиями настоящей Политики. Если вы не согласны с условиями, пожалуйста, не используйте Бот.

2. Информация, которую мы собираем

2.1. Данные, предоставляемые Telegram

При использовании Бота мы получаем следующую информацию от Telegram:

  • Идентификатор пользователя (User ID) — уникальный числовой идентификатор в Telegram
  • Имя пользователя (Username) — если установлен
  • Имя и фамилия — отображаемое имя пользователя
  • Фотография профиля — если пользователь устанавливает её через Бот
  • NFT Username — если пользователь имеет NFT-имя пользователя

2.2. Данные, собираемые при использовании Бота

  • Роль пользователя — статус в системе (персонал, гарант, скамер, нет в базе)
  • Биография — текстовая информация, которую пользователь может установить
  • Статистика — количество занесённых скамеров, выданных спасибо, полученных выговоров
  • История проверок — информация о том, кто и когда проверял пользователя
  • Записи о скамерах — информация о занесённых в базу скамерах (причина, доказательства, дата занесения)
  • Жалобы — текстовые жалобы пользователей (до 1048 символов)
  • Медиафайлы — фотографии профиля, загруженные пользователями

3. Цели использования данных

Мы используем собранные данные исключительно для следующих целей:

  • Предоставление функционала Бота (проверка пользователей, занесение скамеров, управление ролями)
  • Защита пользователей от мошенников и скамеров
  • Ведение статистики и отчётов для персонала
  • Обработка жалоб и апелляций
  • Улучшение работы Бота и исправление ошибок
  • Соблюдение требований законодательства

4. Хранение данных

Все данные хранятся в локальной базе данных SQLite на сервере, где размещён Бот. Данные хранятся до тех пор, пока:

  • Пользователь не удалит свой аккаунт в Telegram
  • Администрация не удалит данные вручную (например, при удалении записи о скамере после апелляции)
  • Не истечёт срок хранения данных согласно внутренним правилам проекта
⚠️ Важно: Мы не передаём данные третьим лицам и не используем их для коммерческих целей.

5. Права пользователей

Вы имеете право:

  • Просмотр данных — запросить информацию о том, какие данные о вас хранятся в Боте
  • Исправление данных — запросить исправление неточных данных (через команду /фото для медиа, через администрацию для других данных)
  • Удаление данных — запросить удаление ваших данных из базы (через апелляцию)
  • Отзыв согласия — прекратить использование Бота в любой момент

Для реализации ваших прав обращайтесь к администрации проекта через канал апелляций или личным сообщением владельцу/заместителю.

6. Безопасность данных

Мы принимаем меры для защиты ваших данных от несанкционированного доступа, изменения, раскрытия или уничтожения:

  • База защищена и доступна только администрации проекта
  • Доступ к серверу ограничен и защищён
  • Регулярное резервное копирование данных
  • Мониторинг подозрительной активности

7. Передача данных третьим лицам

Мы не передаём ваши персональные данные третьим лицам, за исключением случаев:

  • Когда это требуется по закону или по запросу уполномоченных органов
  • Когда это необходимо для защиты прав и безопасности пользователей проекта
  • Когда данные передаются в рамках функционала Telegram Bot API (необходимо для работы Бота)

8. Cookies и аналогичные технологии

Бот не использует cookies или аналогичные технологии для отслеживания пользователей. Все данные хранятся исключительно в базе данных на сервере.

9. Изменения в Политике

Мы оставляем за собой право вносить изменения в настоящую Политику. О существенных изменениях мы уведомим пользователей через новостной канал проекта. Продолжение использования Бота после внесения изменений означает ваше согласие с новой версией Политики.

10. Возрастные ограничения

Бот предназначен для пользователей старше 13 лет (минимальный возраст для использования Telegram). Если вам меньше 13 лет, пожалуйста, не используйте Бот.
